# AI Prompt Store

بوت ذكاء اصطناعي ينتج باقات أوامر ChatGPT متميزة وينشرها على Gumroad.

## المنتجات
- باقات 25-40 أمر ChatGPT جاهزة للنسخ واللصق
- مجالات: تسويق، عقارات، كتابة محتوى، سيو، تجارة إلكترونية، وغيرها
- مع غلاف احترافي + ملف PDF و TXT

## كيف يعمل
1. AI يولد باقة أوامر ChatGPT لمجال محدد
2. ينظمها بفئات وشرح استخدام كل أمر
3. يرفع المنتج على Gumroad بسعر $9-$14
4. يعرضه في متجر GitHub Pages

## الإعداد
1. ضف الـ Secrets في GitHub:
   - `GROQ_API_KEY`
   - `GUMROAD_TOKEN`
2. فعل GitHub Pages من Settings → Pages (Source: GitHub Actions)

## الربح
- سعر المنتج: $9-$14
- إنتاج يومي: 2 منتج
- شهرياً: ~60 منتج في المتجر
- 5-10 مبيعات/يوم = $450-$1400/شهر
